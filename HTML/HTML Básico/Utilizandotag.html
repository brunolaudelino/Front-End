Surgimento da Internet

"O surgimento da internet teve relação com a disputa tecnológica travada entre os Estados Unidos e a União Soviética durante o período da Guerra Fria. No contexto da corrida espacial, os soviéticos lançaram Sputnik — o primeiro satélite artificial — para o espaço. Esse evento fez com que o governo norte-americano desenvolvesse um projeto que resultaria na criação da internet.

1958, surgiu a Advanced Research Projects Agency (Arpa), atualmente conhecida pela sigla Darpa. Essa agência estabeleceu um programa para o desenvolvimento de um sistema que permitisse o envio de informações e dados importantes entre os centros militares e de pesquisa e o Pentágono. Além disso, pensava-se em usá-lo para proteção e defesa do país em caso de uma guerra nuclear.

O projeto em questão realizado pela Arpa foi nomeado de Arpanet e conduzido por Robert Taylor e Lawrence Roberts, e ainda contou com estudos realizados por inúmeros outros cientistas. Na época foi usada a Wide Area Networks (WAN) como tecnologia para a transmissão dos dados.

A Arpanet foi inaugurada em 1969, e, na época, havia apenas quatro computadores nos Estados Unidos capazes de suportar o envio de dados por essa rede. Na ocasião, a mensagem foi enviada de um computador da Universidade da Califórnia, em Los Angeles (UCLA), para outro computador no Instituto de Pesquisa Stanford (SRI), em Menlo Park, também na Califórnia.

O teste inicial seria realizado com o envio da mensagem “log win”, mas o computador em Stanford não suportou a operação e travou quando recebeu a letra “g”. Um segundo teste, realizado momentos depois, teve sucesso. A partir daí, uma conexão permanente se estabeleceu entre UCLA e SRI."

Como funciona o HTML !

HTML, abreviação de HyperText Markup Language, é a linguagem padrão usada para criar e projetar páginas da web. Funciona como uma linguagem de marcação que descreve a estrutura e o conteúdo de uma página da web usando elementos HTML.

Aqui está uma visão geral simples de como o HTML funciona:
  1. Estruturação do Conteúdo: O HTML organiza o conteúdo da página em uma estrutura hierárquica usando elementos como header, `<footer>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, e outros. Esses elementos ajudam a dividir o conteúdo da página em partes significativas.
  2. Elementos e Tags: Os elementos HTML são os blocos de construção básicos de uma página da web. Cada elemento é definido por uma tag HTML. As tags são colocadas entre colchetes angulares e geralmente vêm em pares: uma tag de abertura e uma tag de fechamento. Por exemplo, `p` é uma tag de parágrafo de abertura e `/p>` é uma tag de parágrafo de fechamento.
  3. Atributos: Além das tags, os elementos podem conter atributos, que fornecem informações adicionais sobre o elemento. Por exemplo, a tag `<img>` pode ter o atributo `src` para especificar a origem da imagem e o atributo `alt` para fornecer um texto alternativo para a imagem.
  4. Renderização pelo Navegador: Depois que o navegador recebe o código HTML de uma página da web, ele interpreta o HTML e renderiza a página de acordo com as instruções fornecidas. Isso inclui exibir o texto, formatar o layout, carregar imagens e executar scripts.
  5. Interatividade e Estilo: Embora o HTML seja usado principalmente para estruturar o conteúdo de uma página da web, ele também pode incorporar CSS (Cascading Style Sheets) para estilizar a aparência da página e JavaScript para adicionar interatividade e funcionalidade dinâmica.

Em resumo, o HTML funciona como uma linguagem de marcação que descreve a estrutura e o conteúdo de uma página da web, permitindo que os navegadores interpretem e exibam a página de forma adequada para os usuários.

O que são Tags

Tags em HTML são elementos fundamentais da linguagem de marcação utilizada para criar documentos da web. Elas são usadas para definir diferentes partes do conteúdo e da estrutura de uma página da web. Uma tag consiste em um nome que é colocado entre colchetes angulares "<>"

Existem dois tipos principais de tags em HTML:

  1. Tags de Abertura**: São usadas para indicar o início de um elemento HTML e contêm o nome do elemento. Por exemplo, `<p>` é uma tag de abertura que indica o início de um parágrafo.
  2. Tags de Fechamento**: São usadas para indicar o fim de um elemento HTML e geralmente têm o mesmo nome que a tag de abertura, mas precedido por uma barra "/". Por exemplo, `</p>` é uma tag de fechamento que indica o fim de um parágrafo.

As tags HTML são essenciais para estruturar o conteúdo de uma página da web e fornecer instruções ao navegador sobre como renderizá-la corretamente.
